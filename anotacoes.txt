- Nodejs - Plataforma JS para Servidor que utiliza o protocolo HTTP para fazer receber Requests (Input) e retornar Responses (Output). Também pode se conectar com o banco de dados.
O javascript no navegador opera no contexto de um documento (DOM) e o javascript em um servidor opera no contexto de servidor (Eventos de servidor, variáveis, estruturas de dados etc)


- Heroku - Servidor gratuito

- Node é LTS (Long Term Support)


Nodemon - Roda o servidor novamente sempre que o código muda
